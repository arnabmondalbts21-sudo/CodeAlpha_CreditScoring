# CodeAlpha Credit Scoring Model

This project is part of the CodeAlpha Machine Learning Internship.

## 📌 Objective
To predict whether a person is creditworthy or not using machine learning.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## 📊 Model Used
- Random Forest Classifier

## ⚙️ Features
- Data generation using sklearn
- Data preprocessing (scaling)
- Model training and testing
- Evaluation using:
  - Classification Report
  - ROC-AUC Score
  - Confusion Matrix
